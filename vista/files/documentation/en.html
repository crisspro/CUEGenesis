CUE Genesis

Introduction

CUE Genesis, is a software for generate CUE files to splite a audio track: songs from an album, podcast sections, chapters from an audiobook, etc.

CUE Genesis is developed with GPL 3.0 license.

Visit project site

Requirements

  • Windows 10 or higher, 32 or 64 bits.

How to use it

  1. Load audio file to splite it
  2. Complete album metadata
  3. Play and make a mark where you want.
  4. when you have created all the marks,, press the CUE generate button. The file will be saved in the folder of audio file loaded.
