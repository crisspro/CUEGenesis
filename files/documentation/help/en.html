Labrandeos

Introduction

Labrandeos allows you to split an audio file into different tracks, being able to export them as separate audio files or a single image in CUE format. Songs from an album, sections of a podcast, chapters of audiobooks, among others, are examples of the different situations where Labrandeos can be useful.

System requirements

  • Windows 10 or higher, 32 or 64 bits.
  • Codecs installed on your system for the audio formats you want to play. K-Lite Codec Pack Standard or higher editions are recommended.

Usage

  1. Load the audio file you want to split
  2. Fill in the album metadata
  3. Play the audio track and create a marker right at the point you want
  4. Once you have created all the markers you need, choose whether to export as a CUE image or as separate audio tracks
  5. Press the "export" button to generate the result.

  • When exporting as a CUE image, the resulting image will be saved in the same folder as the original audio.
  • When choosing separate audio tracks, you can configure the output format by clicking the "options" button.

Options

In the tools menu, click on "options" to access the settings of Labrandeos.

General

In this tab, you can select the preferred language, configure export features, as well as options related to Labrandeos updates and select sound alerts for certain actions.

Audio

This section contains the settings for different output formats when exporting as separate tracks.

The default value is set to "automatic." This means Labrandeos will detect the format and other properties of the loaded audio, such as bitrate, bitrate mode, and sampling rate, in order to apply the same values to the exported tracks. If the input format is not supported as an output format, the program will attempt to export in MP3 format with a constant bitrate of 320 kb/s and a sampling rate of 48000 Hz. If you prefer, you can manually choose the output format for the loaded audio.

By checking the "normalize" option, each exported audio track will be normalized to -1.0 dB, meaning the maximum volume of each track will have this value.

By checking the "add silence at the end of each track" option, a two-second silence will be added at the end of each resulting audio file.
