Labrandeos

Introduction

Labrandeos, is a software for export CUE files to split an audio track: songs from an album, podcast sections, chapters of an audiobook, etc.

Labrandeos is developed under the GPL 3.0 license.

Visit project site

System requirements

  • Windows 10 or higher, 32 or 64 bits.

Usage

  1. Load audio file to split it
  2. Complete album metadata
  3. Play and make a mark where you want.
  4. When you have created all the marks, press the "export Cue" button. The file will be saved in the folder where the audio file was loaded.
